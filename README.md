# CS 3502 Project 2: CPU Scheduling Simulator
**Author:** Jordi Padierna Lopez  
**Institution:** Kennesaw State University (OwlTech Industries)  

## Description
A C# Windows Forms application that simulates CPU scheduling algorithms. 
Implemented algorithms include:
* First-Come, First-Served (FCFS)
* Shortest Job First (SJF)
* Shortest Remaining Time First (SRTF) (Preemptive)
* Highest Response Ratio Next (HRRN) (Non-preemptive with aging)

## How to Run
Open the folder in Visual Studio Code with the .NET SDK installed and run `dotnet run`.
